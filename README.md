# Sudoku Solver
This is a sudoku solver using the Backtracking algorithm.

## Getting Started
## Requirements
* `python` 3.x

## Instructions
1. Clone the Repository using the URL
```sh
$ git clone https://github.com/anishsownderraj/Sudoku-Solver
```
## Compiling and Executing Program
```sh
$ python3 Sudoku.py
```
